# Documentasi Instalasi Otomax di Kas Digital

## latar belakang

Otomax adalah sebuah aplikasi server pulsa yang cukup populer di indonesia. Aplikasi ini memungkinkan pengguna untuk mengelola penjualan pulsa, paket data, dan layanan lainnya dengan mudah. Kas Digital adalah salah satu perusahaan yg bergerak di bidang penjualan pulsa dan paket data yang menggunakan Otomax sebagai sistem backend mereka.

## Tujuan

Dokumentasi ini bertujuan untuk memberikan panduan langkah demi langkah tentang cara menginstal dan mengkonfigurasi Otomax di lingkungan Kas Digital. Dengan mengikuti panduan ini, diharapkan pengguna dapat dengan mudah mengatur Otomax dan memulai operasional penjualan pulsa mereka.

## HOW TO GUIDE

project ini hanya berisi penjelasan feature dan step / langkah melakukan konfigurasi otomax dan related add on software lainnya.

## Libraries and Dependencies

- python >= `3.14`
- uv
- mkdocs-material
- mkdocs
- mkdocs-material-extensions

## Installation

project ini bisa di akses public pada repository nya di github atau bisa di install secara lokal.
